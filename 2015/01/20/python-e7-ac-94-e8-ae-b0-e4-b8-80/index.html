Python笔记(一)

(1)str[begin:end:step]以step步数，从begin到end-1。
特殊情况是：
&gt;&gt;str1 = "helloworld"
&gt;&gt;str1[::-1]='dlrowolleh'
(2)raw_input()用来接收我们从console上输入的数，（read-after-write）代码可以这么写
y = float(raw_input(''Please enter a number: ))
上一段代码的意思即提示输入一个数字，并且将数字转换成浮点数，赋值给y
(3)3*'a'表示三个'a'，即'aaa'
(4)elif即别的编程语言中的else if
(5)在Python规范里，比较String和int大小时，String总比int大。
(6)用Python解决了一个很经典的数学题（囧o(╯□╰)o）：某次数学竞赛共有10道选择题,每答对一题得4分,不答对得0分,答错扣1分，一共可能有多少种可能的成绩？
代码贴上来，

for x in range(0,11):     for z in range(0,11-x):         y = 10-x-z         fx = 4*x - y         if fx not in arr:             arr.append(fx) print(len(arr))

Python的数组下标很坑，一定要注意，range里亦是。
但是Python的in …这种判断方法，可以用来做很多事，比如将数组去重。
(7)“guess and check” methods
即猜测初始值，然后逐渐逼近的思想。比如求一个整数的平方根或者立方根时，先初始化，再逐渐逼近到一定的值，就可以了。
代码来自MIT6.00.1课程：

x = int(raw_input('Enter an integer: ')) for ans in range(0, abs(x)+1): if ans**3 == abs(x): break if ans**3 != abs(x): print(str(x) + ' is not a perfect cube') else: if x &lt; 0: ans = - ans print('Cube root of ' + str(x) + ' is ' + str(ans))
当没有初始值时，Exhaustive enumeration穷尽枚举法在有序且有限集合里很有效。
(8)Python里break也是跳出最近层循环
